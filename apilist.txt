Jokes APIs
    https://api.chucknorris.io/
        API Link: https://api.chucknorris.io/jokes/
        Example response:
            {
                "icon_url" : "https://assets.chucknorris.host/img/avatar/chuck-norris.png",
                "id" : "1fQ9opiATFKTEtte8-Jthg",
                "url" : ""
                "value" : "Some people say that god can walk on water. Chuck Norris can swim on land"
            }

        Retrieve a random chuck norris joke from a given category.
            https://api.chucknorris.io/jokes/random?category={category}

        Retrieve a list of available categories.
            https://api.chucknorris.io/jokes/categories

        Free text search.
            https://api.chucknorris.io/jokes/search?query={query}



    https://sv443.net/jokeapi
        API Link: https://sv443.net/jokeapi/category/{Category}

        Example response:
            {
                “category”: “Programming”,
                “type”: “single”,
                “joke”: “I’ve got a really good UDP joke to tell you but I don’t know if you’ll get it.“,
                “id”: 0
